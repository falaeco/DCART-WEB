Description English:

The project consists of a puzzle/game that functions based on natural forces that provoke movement. My concept is inspired by motion, gravity, and impulse. Puzzle Impulso is meant to be a game that is built using different curved pieces of plywood which can take different configurations. adepending on these configurations a small ball thrown from the starting point can take different velocities when following the path determined by the user. It is meant to be interesting and fun.



----

Description French:

Le projet consiste en un puzzle/jeu qui se base sur les forces naturelles qui provoquent le mouvement. Mon concept s’inspire du mouvement, de la gravité et de l’impulsion. Appelé Puzzle Impulso, c’est un jeu construit à l’aide de différentes pièces courbes en contreplaqué qui peuvent prendre différentes configurations et, selon celles-ci, une petite balle lancée depuis un point de départ peut prendre différentes vitesses en suivant le chemin déterminé par l’utilisateur. Il est conçu pour être intéressant et amusant.
